# UX Kit Base v1.0

UX Kit Base es el sistema de fundamentos de **UX Kit Express**.  
Define reglas claras para construir interfaces profesionales sin saber diseño.

## Principios de Diseño

1. Claridad antes que estética  
2. Consistencia predecible  
3. Feedback inmediato  
4. Accesibilidad por defecto  
5. Escalabilidad sin fricción  

## Convenciones de Naming

- Semántico y funcional  
- `.btn-primary`, `.card-header`, `.input-error`
- Prefijos:
  - `ux-` layout
  - `is-` estados JS (`is-loading`)

## Why this is good UX

- Elimina decisiones arbitrarias  
- Reduce deuda visual  
- Garantiza legibilidad y contraste  
- Permite aprender UX mientras se desarrolla
